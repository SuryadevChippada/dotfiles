# My Dotfiles

This repository contains all my personal configuration files (dotfiles) for various apps and tools.  
Keeping them in GitHub allows me to quickly set up a new machine or restore my preferred environment.

## What's Inside

- **Shell & Terminal**
  - `.zshrc` – Zsh configuration
  - `.wezterm.lua` – WezTerm terminal configuration
  - `alacritty.toml` – Alacritty terminal configuration
  - `starship.toml` – Starship prompt configuration

- **Editors & IDEs**
  - `nvim/` – Neovim configuration
  - `textstudio/` – Text editor setup

- **System & Utilities**
  - `fastfetch/` – System info tool config
  - `mufetch/` – Another system fetcher
  - `neofetch/` – System fetcher
  - `raycast/` – Raycast extensions
  - `wtf/` – Custom scripts

- **Other Tools & Apps**
  - `bagels/`, `broot/`, `btop/`, `cava/` – CLI tools
  - `ghostty/` – Terminal web client
  - `github-copilot/` – Copilot config
  - `gophertube/` – Personal Go-based video tool
  - `jrnl/` – Journal CLI tool
  - `stormy/` – Misc configs
  - `xbuild/` – Build tools
  - `yazi/` – Misc scripts
  - `fontconfig/conf.d/` – Font configurations
  - `configstore/update-notifier-@google/` – App notifications

- **Other**
  - `macmon.json` – System-specific config file
  - Various helper scripts and settings for CLI and GUI tools
